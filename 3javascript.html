JavaScript

JavaScript는 종종 JS로 약칭되며 고급스럽고 동적이며 약한 형식의 프로토타입 기반, 다중 패러다임 및 해석 프로그래밍 언어입니다. HTML 및 CSS와 함께 JavaScript는 월드 와이드 웹 콘텐츠 제작의 세 가지 핵심 기술 중 하나입니다. 웹 페이지를 대화식으로 만들고 비디오 게임을 포함한 온라인 프로그램을 제공하는 데 사용됩니다. 대부분의 웹 사이트는이를 사용하며 모든 최신 웹 브라우저는 내장 된 JavaScript 엔진을 통해 플러그인 없이도 지원합니다.
